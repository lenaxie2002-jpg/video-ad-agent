# 1. 吊坠灯 / 小吊灯：岛台厨房场景

## 1.1 总体风格方向

岛台厨房是吊坠灯 / 小吊灯的核心应用场景之一。整体风格偏向：

- 高端自然厨房岛台
- Arhaus 风
- 法式复古奶油风
- California-European quiet luxury
- 加州自然风
- 现代自然风
- 现代 farmhouse
- 英式古典
- 黑白木色厨房
- 轻奢白金厨房
- 真实室内摄影感
- 温暖通透、自然日光、生活化但干净高级

## 1.2 空间元素

可使用的厨房空间元素包括：

- 奶油白墙面
- 暖白墙顶
- 浅木 / 白橡木整墙柜
- 白色平板柜门
- 开放木搁板
- 浅灰 / 白色大理石挡板
- 强纹理灰白大理石挡板
- zellige / 手工砖墙
- 细长手工砖
- 白色石膏烟机罩
- 拱形烟机罩
- 黑色烟机罩
- 拱形门洞
- 拱形厨房框景
- 浅木梁
- 旧木梁
- 黑框窗
- 大面积自然日光
- 藤编罗马帘 / 竹帘
- 浅木地板
- 不锈钢冰箱
- 黄铜水龙头
- 黑色水龙头
- 黄铜 / 黑色五金

## 1.3 岛台类型

岛台可以不局限于传统长方形，可包括：

- 长方形岛台
- 圆角岛台
- 弧形岛台
- reeded 竖纹木岛台
- 浅木岛台
- 胡桃木岛台
- 深木岛台
- 黑色 / 炭黑岛台
- 白色瀑布岛台
- 白色石英石台面岛台
- 灰白大理石台面岛台
- 浅木 + 白色石英石 / 大理石组合岛台

## 1.4 吧椅类型

可搭配：

- 天然藤编吧椅
- 皮革编织吧椅
- 编织 / 藤编椅
- 黑色细腿吧椅
- 焦糖皮革吧椅
- 透明亚克力 + 黄铜吧椅
- 羊羔绒圆凳
- 木质吧椅
- 现代黑白木色吧椅

## 1.5 吊坠灯类型

岛台吊坠灯可使用：

- 极细黑线小球灯，三头线性排列
- 浅藤编 / 天然藤罩小吊灯，三头排列
- 黑金复古锥形玻璃罩吊灯
- 白色圆盘 + 半球吊灯
- 黑色大锥形金属吊灯
- 白色锥形吊灯
- 带链条的玻璃灯笼吊灯
- 双点固定链条横跨式乳白球灯
- 透明玻璃复古灯笼
- 小乳白球吊坠灯，双头 / 三头线性排列
- 黄铜圆柱小吊灯
- 白色大球吊灯
- 三头小白锥形吊灯
- 白色布罩 + 黄铜链条吊灯
- 透明玻璃方形灯笼 / 玻璃盒吊灯
- 白色半圆金属罩 + 金色内胆吊灯
- 金色稻穗 / 麦穗 / 扇形编织感吊灯
- 黑色方框灯笼
- 黑色圆筒玻璃灯笼
- 黑色锥形链条灯
- 黑金圆锥灯
- 黑色 + 水晶珠链吊灯

## 1.6 灯型与场景匹配

- 细线 / 细杆小吊灯：适合轻盈现代岛台
- 链条 / 灯笼吊灯：适合法式复古或英式古典厨房
- 藤编灯：适合自然度假感 / 加州风厨房
- 黑色锥形灯：适合现代乡村或黑白木色厨房
- 小球 / 小圆筒灯：适合轻盈现代厨房
- 布罩 / 灯笼 / 编织灯 / 黑锥灯：适合更宽岛台或挑高空间
- 黄铜圆柱小吊灯：适合黑白石材厨房
- 白色大球吊灯：适合现代极简大理石岛台
- 三头小白锥形吊灯：适合透过拱形门洞做居中构图

## 1.7 安装规则

后续岛台厨房 prompt 必须强调：

- 吊坠灯沿岛台长轴线性排列
- 双头 / 三头 / 多头必须等距、居中
- 与岛台长度和吧椅数量匹配
- 与水槽、灶具、吧椅数量、岛台中心线对齐
- 灯具底部明显高于台面
- 不遮挡视线
- 不压低空间
- 灯具比例随岛台长度调整
- 不可过低、过密或偏离岛台中心线

## 1.8 软装元素

台面软装可使用：

- 木碗水果
- 柠檬、梨、蔬菜
- 陶瓷花瓶
- 自然枝条
- 鲜花
- 紫色 / 白色花艺
- 料理书
- 砧板
- 面包
- 碗盘
- 玻璃杯
- 手工陶罐
- 烛台
- 玻璃蛋糕罩
- 透明罐
- 亚麻布

软装应生活化但整洁高级。

## 1.9 色彩范围

可使用：

- 奶油白
- 暖白
- 浅橡木
- 蜂蜜木
- 胡桃木
- 黑色 / 炭黑
- 黄铜
- 灰白石材
- 鼠尾草绿
- 陶土色
- 焦糖皮革
- 天然藤色

## 1.10 避免项

避免：

- 灯具过大过低
- 线材过粗
- 排列不齐
- 灯具悬浮
- 不符合岛台中心线
- 厨房太空
- 商业吧台感
- 廉价样板间
- 过度冷白或灰色极简
- 过冷灰
- AI 感强
- 吊线歪斜
- 灯具与岛台不匹配

- 拱形结构

- 亚麻窗帘

## 4.4 浴缸上方灯具规则

- 吊灯 / 小吊灯 / 半吸顶灯需居中于浴缸上方
- 或居中于浴缸与窗户中心线
- 比例克制
- 灯底不能过低接近水面
- 不能压迫浴缸
- 不能遮挡窗景

可用灯型：

- 乳白玻璃球
- 黄铜 / 黑色小吊灯
- 藤编 / 天然编织灯
- 布艺褶皱半吸顶

- 链条灯笼
- 白色圆盘 / UFO 灯
- 古铜半吸顶

## 4.5 镜前灯具规则

- 左右对称壁灯或成对小吊坠
- 贴近镜子两侧
- 与镜子高度和台盆中心对齐
- 灯体细长 / 小巧
- 不能挡住镜面和龙头
- 双台盆可用两镜两侧壁灯
- 双台盆也可用三盏吊坠
- 双台盆也可用单镜左右对称
- 五金颜色需与龙头、柜门拉手、镜框呼应

## 4.6 风格分支

可做：

- 明亮自然的奶油浴室
- 加州自然浅木浴室
- 法式复古粉房
- 英式 / 复古图案壁纸浴室
- 黑色大理石或深色粉房氛围款
- 带浴缸和大窗的精品酒店浴室
- 洗衣房 / 水槽区小吊灯场景

## 4.7 画面要求

- 真实自然光
- 温暖通透

## 9.5 材质标注规则

同一张图中：

- 如果已经标注具体金属材质，例如 BRUSHED BRASS、STAINLESS STEEL、SOLID BRASS

- 优先保留更具体准确的材质名称

## 9.6 画面洁净度规则

后续生成或修改功能卖点标注图时：

- 背景和整体画面必须干净无杂质
- 无脏点
- 无噪点
- 无斑驳污渍
- 无多余纹理
- 无异常阴影

- 灯具主体、水晶 / 玻璃和金属表面也不要出现多余污点、黑点、瑕疵或 AI 杂质

# 家具基础空镜头视频生成模板

本文档用于家具类产品图 / 场景图生成写实风格视频片段，重点控制：  
**家具结构不变、布局不变、比例不变、光影自然、运镜克制、画面高级真实。**

---

# 一、基础空镜头模板

## 1. 通用基础模板

适用于所有家具视频生成任务，建议每次都加入。

```text
基于提供的图片生成一段写实风格视频。
保持所有家具、布局、比例、光影风格、构图完全一致。
不允许改变任何物体的位置、结构或尺寸。
画面稳定、真实。
不允许出现变形、闪烁或物体变化。
```

---

## 2. 运镜模块

根据视频需求选择其中一种运镜方式。

### 2.1 固定镜头

```text
镜头保持完全静止。
```

适合：

- 商品页稳定展示
- 材质和造型已经很清楚的家具图
- 不希望 AI 产生结构变化的画面

---

### 2.2 轻微推进

```text
镜头缓慢向前 / 向后推进，运镜轻微且克制，具有电影感。
```

适合：

- 商品页视频
- 广告开头
- 展示家具整体质感

---

### 2.3 轻微横移

```text
镜头进行非常轻微的横向、上升或下降移动，幅度很小。
```

适合：

- 展示空间层次
- 展示家具与背景的关系
- 展示自然光下的材质变化

---

### 2.4 轻微环绕

```text
镜头进行向左 / 向右 / 向下 / 向上的环绕移动，
角度控制在 5–10 度以内，动作缓慢稳定。
```

适合：

- 柜体
- 桌子
- 椅子
- 床头柜
- 有立体结构的家具

注意：

- 不建议做大幅度旋转
- 不建议做 360 度环绕
- 结构复杂的家具应谨慎使用

---

## 3. 光影要求

```text
自然光从窗外照入，光线柔和真实。
随着镜头向左环绕，家具表面的木纹、纹理、凹凸结构和材质细节在光影下产生自然变化。
```

特写时强调：

- 明暗层次
- 材质的细节纹理
- 光泽变化
- 高级、克制、真实的家居广告氛围

禁止：

- 不要过曝
- 不要过强闪烁
- 不要突然变亮或变暗

---

## 4. 视频节奏建议

建议时长：**8 秒**

| 时间段 | 内容 |
|---|---|
| 前 2 秒 | 正面整体静态展示家具 |
| 中间 2–3 秒 | 镜头缓慢向左环绕 |
| 接着 2 秒 | 切入材质细节特写，展示木纹 / 五金 / 边角工艺 |
| 最后 1–2 秒 | 停留在左侧稍偏正面的高级角度，展示整体造型和质感 |

---

## 5. 整体风格

```text
高端家具品牌广告风格。
真实家居环境。
自然光影。
材质细节清晰。
空镜头展示。
高级、安静、克制、有质感。
```

关键词：

- 高端家具品牌广告风格
- 真实家居环境
- 自然光影
- 材质细节清晰
- 空镜头展示
- 高级
- 安静
- 克制
- 有质感

---

## 6. 防翻车限制模块

强烈建议每次都加入。

```text
不允许闪烁。
不允许突然出现。
不允许物体变形。
不允许物体变化。
不允许比例变化。
不允许镜头抖动。
所有变化必须自然、稳定、真实。
```

# 二、光影空镜头模板

## 1. 通用基础模板

```text
基于提供的图片生成一段写实风格视频。
保持所有家具、布局、比例、光影风格、构图完全一致。
不允许改变任何物体的位置、结构或尺寸。
画面稳定、真实。
画面构图保持与原始图片一致。
```

---

## 2. 镜头选择

### 2.1 固定镜头 / 静态空镜头

```text
保持固定，为静态空镜头。
机位不移动，不推拉，不平移，不旋转。
```

适合：

- 光影流动视频
- 材质展示视频
- 最稳妥的商品页视频
- 防止家具结构变形的场景

---

### 2.2 轻微推进

```text
镜头缓慢向前 / 向后推进，运镜轻微且克制，具有电影感。
```

---

### 2.3 轻微横移

```text
镜头进行非常轻微的横向、上升或下降移动，幅度很小。
```

---

### 2.4 轻微环绕

```text
镜头进行向左 / 向右 / 向下 / 向上的环绕移动，
角度控制在 5–10 度以内，动作缓慢稳定。
```

---

## 3. 视频重点表现

材质表现选择其中一个即可。

### 3.1 家具材质

```text
家具表面的木纹 / 金属 / 玻璃 / 大理石 / 洞石 / 纹理。
```

### 3.2 柜门肌理

```text
柜门的竖向凹凸肌理。
```

### 3.3 木质光泽

```text
木质表面的细腻材质感与光泽变化。
```

### 3.4 自然光明暗变化

```text
自然光在柜体表面缓慢移动时形成的明暗层次变化。
```

### 3.5 墙面与软装光影

```text
墙面、柜体、窗帘上的柔和光影流动。
```

---

## 4. 光线效果要求

```text
阳光以自然、缓慢、柔和的方式移动。
画面中的光影缓慢扫过柜门表面。
柜子的竖向纹理随着光线变化呈现更清晰的立体感。
光影变化要真实、细腻、有层次。
不要过强闪烁，不要突然变亮或变暗。
```

---

## 5. 整体氛围

```text
高端家具品牌广告风格。
真实家居环境。
温暖自然光。
安静、克制、质感高级。
强调材质细节和光影美感。
```

---

## 6. 不允许出现

```text
不允许镜头移动。
不允许柜体变形。
不允许材质变化。
不允许装饰物移动。
不允许闪烁跳帧。
不允许光影不自然。
不允许画面穿模。
不允许结构变化。
```

注意：  
如果选择“固定镜头 / 光影空镜头”，则镜头应保持不动，所有变化只体现在自然光和材质受光变化上。

---

## 7. 允许变化范围

```text
所有变化只体现在：
自然光的缓慢移动、
柜体表面纹理的受光变化、
空间光影的细腻流动。
```

---

## 8. 视频节奏建议

建议时长：**6–8 秒**

| 时间段 | 内容 |
|---|---|
| 前 2 秒 | 静态展示柜体整体 |
| 中间 3–4 秒 | 自然光缓慢移动，光影扫过柜体表面 |
| 最后 2 秒 | 停留在最能体现木纹和立体肌理的光影状态 |

---
